### nhcap
python undetected_chromedriver 的自动解决hcaptcha
###支持的类型
|类型         | 支持情况                        | 模型训练                    |  缺点                                         |优点                                     |
|   -------   |             -------             |           -------           |                    -------                    |                 -------                 |
|九宫格点选    | 需要对应的yolov5模型(支持中英文)  | 训练容易                     |hcaptcha更新较快一个模型可能只能支持几小时        |训练需要配数据集少(80张图片)，绝大多数可训练 |
|图片选择位置  | 需要对应的yolov5模型(仅支持中文)  | 训练需要相对较多的数据集       |需要的训练数据较多，hcaptcha更新未测试，预估(7天) |坚持时间长                                |
